# rest

A collection of useful and/or interesting resources about REST.

## Methods

- GET `/items[/:id]`
  - Respond with a list of items, or a single item with the matching ID.
- POST `/items`
  - Create a new item.
- PUT `/items/:id`
  - Make change to an existing item.
- DELETE `/items/:id`
  - Remove the item with the provided ID.
